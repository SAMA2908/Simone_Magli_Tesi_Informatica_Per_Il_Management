# Simone_Magli_Tesi_Informatica_Per_Il_Management
La tesi confronta la Regressione Lineare Multipla, il Random Forest e l'SVR (tuned) per predire il GPA basandosi sulle abitudini di vita (studio, sonno, stress).  Risultato chiave: le ore di studio sono l'unico forte predittore. Contro le aspettative, il modello più semplice (Regressione Lineare) è risultato il più stabile e performante.
